Why HTML and CSS are enough for the Shàn viewer

No scripts. No trackers. Just structure and style.

The folding fan opens in stages. Likewise, a page can reveal meaning through typography and color before any code runs on the client — because none does.

Shàn programs are written in tags, but this viewer only renders static HTML and CSS folders. Point it at viewer/samples, not at examples/*.shan.

What you can build

Documentation, portfolios, reports, and styled forms that submit via plain HTTP — all valid in this viewer.
